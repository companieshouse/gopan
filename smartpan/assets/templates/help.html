Using SmartPAN with getpan

getpan natively supports SmartPAN.

Pass the SmartPAN mirror with the -smart command line parameter:

getpan -smart http://path.to/SmartPAN

This should allow getpan to locate any version of any module from any of the available indexes.

Using SmartPAN with cpanm or Carton

cpanm and Carton will use the 02packages.details.txt file to locate modules.

By default, SmartPAN will list the most recent version of each module in the 02packages file.

This is per-index, so accessing SmartPAN using the CPAN or BackPAN URLs will typically find different module versions.

The SmartPAN index will list the most recent version of a module available in any of the other indexes. Using the SmartPAN URL should consistently find the latest module version.

cpanm --mirror http://your.smartpan/SmartPAN Some::Module
cpanm --mirror http://your.smartpan/SmartPAN --install-deps .
PERL_CARTON_MIRROR=http://your.smartpan/SmartPAN carton install
